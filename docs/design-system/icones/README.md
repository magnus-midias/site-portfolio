# Icones

Pasta para os ícones do projeto.

## Formatos aceitos

- `.svg` — preferencial (inline ou como asset).
- `.png` — somente quando SVG não for viável.

## Convenção de nomenclatura

Sem acentos, sem caracteres especiais, kebab-case. Prefixo `ic-` para diferenciar de logos. Exemplos:

```
ic-seta-direita.svg
ic-menu.svg
ic-fechar.svg
ic-check.svg
```

## Tamanhos de referência

Exportar em viewport 24×24 px (padrão) ou conforme necessidade do projeto.
Documentar aqui se o projeto usar um tamanho diferente.
