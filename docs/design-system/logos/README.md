# Logos

Pasta para os logotipos da marca em todas as variações.

## Formatos aceitos

- `.svg` — preferencial (vetorial, escalável).
- `.png` — para contextos que não suportam SVG (fundo transparente, mínimo 512×512 px).
- `.webp` — opcional, para uso em web com fundo.

## Convenção de nomenclatura

Sem acentos, sem caracteres especiais, kebab-case. Exemplos:

```
logo-principal.svg
logo-horizontal.svg
logo-variacao-escura.svg
logo-variacao-clara.svg
logo-simbolo.svg
```

## Organização

Se houver muitas variações, criar subpastas por tipo:
```
logos/
  svg/
  png/
```
