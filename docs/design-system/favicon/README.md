# Favicon

Pasta para os arquivos de favicon em todos os formatos necessários.

## Formatos e tamanhos recomendados

| Arquivo | Tamanho | Uso |
|---------|---------|-----|
| `favicon.ico` | 16×16, 32×32 | Compatibilidade universal |
| `favicon-32x32.png` | 32×32 | Navegadores modernos |
| `favicon-16x16.png` | 16×16 | Abas do navegador |
| `apple-touch-icon.png` | 180×180 | iOS / Safari |
| `android-chrome-192x192.png` | 192×192 | Android / Chrome |
| `android-chrome-512x512.png` | 512×512 | PWA splash screen |
| `site.webmanifest` | — | Manifesto do PWA |

## Convenção de nomenclatura

Sem acentos, sem caracteres especiais, kebab-case (seguir nomes da tabela acima como padrão).

## Ferramentas recomendadas

Gerar os formatos a partir de um SVG ou PNG de alta resolução usando realfavicongenerator.net ou ferramenta equivalente.
